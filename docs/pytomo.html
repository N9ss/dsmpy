pytomo

pytomo is a Python tool for seismic tomography based on pydsm. pydsm can compute several seismic models in parallel, which is used in pytomo to run probabilistic waveform inversion with a neighbourhood algorithm (Sambridge, 1999).

The video below show an example of synthetic inversion with a neighbourhood algorithm for the depth and shear-velocity jump in the D'' layer at the bottom of the Earth's mantle (left: voronoi cells colored by waveform variance; middle: target, and best inverted model for the current iteration; right: waveforms for the target, and current best inverted model)
