pydsm

pydsm is a Python tool for exact computation of seismic waveforms in spherically symmetric (1-D) transversely isotropic (TI) media using the direct solution methods (DSM). pydsm is based on the Fortran codes of the DSM (kawai et al., 2006) compiled into a python library (using numpy.f2py).

pytomo

pytomo is a Python tool for seismic tomography based on pydsm. pydsm can compute several seismic models in parallel, which is used in pytomo to run probabilistic waveform inversion with a neighbourhood algorithm (Sambridge, 1999).
